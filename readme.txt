Create more immersive WordPress image galleries by turning any images into virtual reality shots.

== Description ==

This virtual reality WordPress plugin will help you make standard flat image galleries more innovative and creative transforming them into stylish VR technology content. 

To build a virtual reality photo gallery, you'll need to add a shortcode with your custom attributes (no tech knowledge is needed) and choose images. This will instantly bring new virtual reality experience to your website. 

You don't need to have any special gear for taking shots and creating VR galleries, every image you choose will be treated as VR-shot and can be moved around by site visitors. 

Image galleries built with VR Gallery will help you create more visual promotion for your content, evoke curiosity and increase overall user engagement on your site. More fun, more interaction. 

The plugin provides perfect PC, mobile and tablet navigation experience. 

== Installation ==

1. Install the plugin via FTP or from Dashboard.
2. Activate plugin.
3. Add gallery shortcode from TinyMCE panel.
